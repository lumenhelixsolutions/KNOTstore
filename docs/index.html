# Round-trip: encode → decode → retrieve
from knotstore.codec import encode_manifest, decode_manifest
from knotstore.knotstore import KnotStore

store = KnotStore()
ptr   = store.put(b"hello world")

manifest = encode_manifest(store, ptr)    # bytes
m2       = decode_manifest(manifest)        # reconstruct
data     = store.get(m2.pointer)          # O(1) lookup — exact address
assert data == b"hello world"              # ✓

Version history

Iterative correction

Each version fixed a specific, measurable problem in the draft — documented rather than papered over.

v0.1.1
Address-regenerating retrieval
The draft's get() linearly scanned the entire backend — O(N). Fixed: address_for() is the single source of truth for both put and get.
v0.1.2
1-byte binary pointer + content placement
JSON pointer drifted to 186 bytes. codec.py reduces it to 1 byte (186× compression). SimHash content placement wired end-to-end.
v0.1.3
Reversible MacroCube + ProvenanceLog
Previously ρ-moves were never applied. cube.py implements a real 27-subcube / 162-face state. Rollback-capable provenance accumulator.
v0.1.4
Braid routes · Cauldron · phase-duality audit
Alexander braid representation of ρ-move routes in B₉. Cauldron canonical semantics. Two-phase audit log (forward p=0, dual p=1).
v0.1.5 — current
Burau matrices · Alexander polynomials · KnotInfo verification
Full reduced Burau representation. Alexander polynomial Δ(t) verified for trefoil, figure-eight, cinquefoil against KnotInfo. All 7 KNOTS_V01 properties confirmed.

Honest scope

What works. What doesn't. Both documented.

Three limitations are stated rather than papered over.

Real and working

  • Deterministic content-addressed store
  • O(1) address-regenerating retrieval (not O(N) scan)
  • Open-address collision recovery (testable with address_bits knob)
  • Exact-duplicate collapse
  • Merkle-root tamper detection
  • 1-byte binary tiny pointer (186× vs JSON)
  • Content-correlated placement (~9× near-duplicate locality)
  • Reversible MacroCube — rollback-capable provenance
  • Burau matrices + Alexander polynomials (verified against KnotInfo)
  • 36 tests — all passing — stdlib only

Still namespacing, not topology

  • δ-channels: assigned by digest[0] % 4, no knot-theoretic meaning
  • Knot labels: a coarse projection of the content signature
  • Real locality lives at the shard level, not the knot coordinate
  • Route Alexander polynomials: zero (route_to_braid is ad hoc)
  • 10_83 confirmed non-invertible — breaks reversibility invariant
  • 10_85 probable non-invertible — same issue
  • MacroCube is reversible but not a security primitive
